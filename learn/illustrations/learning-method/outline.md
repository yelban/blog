# Illustration Outline

## Illustration 1
**Position**: 第一招：錯題不能白錯 / 「高手相反。專挑不會的練，雖然痛苦，但進步快。」之後
**Purpose**: 視覺化「刻意練習」概念，對比高手與一般人的練習方式
**Visual Content**:
- 左側：一般人 → 重複做會的題目，舒適圈內轉圈，進步緩慢（小小的向上箭頭）
- 右側：高手 → 專攻不會的題目，跳出舒適圈，快速進步（大大的向上箭頭）
- 中間分隔線，兩側用顏色區分

## Illustration 2
**Position**: 第二招：分清楚「真會」和「假會」 / 「能這樣問自己的人，進步比別人快。」之後
**Purpose**: 視覺化「確定/不確定」標記的概念
**Visual Content**:
- 左側：傳統方式 → 答對打勾，但有些是矇的（打勾旁有問號）
