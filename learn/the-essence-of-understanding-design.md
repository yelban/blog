# 視覺風格規範：你真的懂了嗎？

## 設計風格

**Swiss Modernism 2.0 + Editorial Magazine**

- 大量留白，聚焦內容
- 精心排版的引用區塊
- 圖片作為視覺節奏點
- 首字放大（Drop Cap）裝飾
- 標題前珊瑚色圓點裝飾

## 字型系統

| 用途 | 字型 | 備註 |
|------|------|------|
| 標題 | Libre Bodoni | 典雅襯線體，知性氣質 |
| 內文 | Noto Sans TC | 台灣繁體，清晰易讀 |
| 程式碼 | SF Mono / Fira Code | 等寬字型 |

## 色彩系統

### 亮色模式（Light Mode）

亮色模式為預設閱讀體驗，強調溫暖舒適的閱讀氛圍。

| 名稱 | 色碼 | 用途 | 對比度 |
|------|------|------|--------|
| 霧藍 | `#7ECFC0` | Primary - 引用框邊線、列表標記、連結 | — |
| 珊瑚 | `#E8756D` | Accent - 強調文字、首字放大、裝飾元素 | 4.5:1+ |
| 玫瑰粉 | `#D4A5A5` | Secondary - 輔助色 | — |
| 象牙白 | `#FEFBF8` | Background - 主背景（近白，微暖） | — |
| 米白 | `#F7F3EF` | Background Alt - 引用區塊、程式碼背景 | — |
| 深炭灰 | `#2D3436` | Text - 正文 | 12:1+ |
| 灰褐 | `#636E72` | Text Muted - 次要文字、圖說 | 4.5:1+ |
| 淺灰 | `#E8E4E1` | Border - 分隔線 | — |

**亮色模式設計原則：**
- 背景使用象牙白 `#FEFBF8`，近白但帶微暖色調
- 正文使用深炭灰 `#2D3436`，對比度 12:1 以上
- 強調色珊瑚 `#E8756D` 用於視覺焦點
- 引用框使用霧藍左邊框，清新不搶眼

## 排版規範

| 屬性 | 數值 |
|------|------|
| **內文字體大小** | **18px（桌面+手機）** |
| 內容最大寬度 | 68ch |
| 行高 | 1.8 |
| 字間距 | -0.01em |
| 標題字間距 | -0.02em |
| 段落間距 | 1.6rem |
| 章節間距 | 4rem |

## 圖片處理

- 全寬展示，突破內容區域
- 圓角：12px（桌面）/ 0（手機）
- 陰影：`0 8px 30px rgba(0, 0, 0, 0.06)`
- 圖說：0.85rem，斜體，居中

## 特殊元素

### 引用區塊（Blockquote）

- 左邊框：4px 霧藍
- 背景：米灰
- 圓角：0 12px 12px 0
- 裝飾引號：4rem 霧藍半透明

### 重點概念區塊（Key Concept）

- 背景：霧藍+珊瑚漸層（10% 透明度）
- 邊框：1px 淺灰
- 圓角：12px
- 內距：2rem

### 首字放大（Drop Cap）

- 字型：Libre Bodoni
- 大小：3.5rem
- 顏色：珊瑚
- 浮動：左

## 互動元素

### 閱讀進度條

- 位置：頂部固定
- 高度：2px
- 顏色：珊瑚

### 回到頂部按鈕

- 位置：右下角固定
- 大小：48px 圓形
- 出現時機：滾動超過 500px

### 動畫效果

- 頁面載入：fadeInUp，分層延遲
- 圖片：滾動進入視窗時淡入上移
- 過渡時間：0.6s ease

## 響應式斷點

| 斷點 | 調整 |
|------|------|
| ≤768px | 字體維持 18px、內距縮減、圖片無圓角 |
| ≥900px | 圖片延伸至 -4rem |

## 主題切換

- 頁首右上角提供切換按鈕
- 亮色模式顯示 🌙 月亮圖示（點擊切換到深色）
- 深色模式顯示 ☀️ 太陽圖示（點擊切換到亮色）
- 首次造訪依照系統偏好設定
- 手動切換後記住選擇（localStorage）
- 背景色切換時有 0.3s 過渡動畫

## 無障礙考量

- 顏色對比度 ≥ 7:1（亮色模式正文達 12:1）
- 所有互動元素有 focus 狀態
- 圖片皆有 alt 描述
- 支援手動切換亮色/深色模式
- 支援 `prefers-color-scheme` 系統偏好
- 列印樣式優化
